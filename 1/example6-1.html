Javascript의 onmouseover와 onmouseout을 사용하면 마우스를 올렸을 때와
내렸을 때의 변화를 다르게 해줄 수 있음.

prompt("메시지", "기본 입력값"): 사용자에게 문자열을 입력아 리턴
자바스크립트는 세미클론이 없어도 동작은 하지만 문장 구분을 위해 사용 권장,
한줄에 여러 문장을 넣을 시 세미클론으로 구분 해야함.
자바스크립트는 따옴표, 쌍따옴표 구분이 없음 (아무거나 써도 상관 없음)
자바스크립트는 모든 변수의 자료형을 var, let로 선언하며,
변수에 저장되는 값에 대한 제약은 없음.
지역 변수와 전역 변수 구분을 위해 this 사용 (this는 전역 변수 임을 알림)
1234E-4는 10의 -4제곱으로 0.1234
null은 값이 없음을, NaN은 수가 아님을 뜻함
""안에 다시 문자열 값을 넣어야 할 경우 '' 사용
>>>는 >>와 달리 부호가 없음
